# Cursor archive (exported)

This folder holds **copies** of Cursor plans and agent transcripts from your machine so they can live in Git.

- **Source:** `~/.cursor/plans/`, `~/.cursor/projects/.../agent-transcripts/`, `uploads/`
- Cursor does **not** automatically read these files; they are only for your records and sharing.

**Public repo warning:** Transcripts and chats may include pasted tokens, emails, or paths. **Search for secrets** (`ghp_`, `sk-`, passwords) before pushing, or keep the repo private.

To refresh this export, re-run the copy commands (or ask the agent to sync again).

**Note:** Uploads that contain third-party reference code (for example TREK) are intentionally **not** mirrored here; keep that reference only in your local `TREK_alt/` tree per `AGENTS.md`.
